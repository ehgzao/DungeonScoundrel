# Dungeon Scoundrel v1.4.0 - Tutorial Update

Released: November 9, 2025  
Play Now: dungeonscoundrel.netlify.app

## What's New

### Complete Interactive Tutorial System

This update introduces a comprehensive 13-step tutorial specifically designed for first-time players on Easy difficulty. New players will now receive professional onboarding that teaches:

- Health System - Understand HP management and healing
- Gold & Economy - Learn resource management
- Merchant Access - Discover the shop system
- Combat Mechanics - Master weapon equipping and fighting
- Room Drawing - Core dungeon exploration mechanic
- Card Types - Differentiate Monsters, Weapons, Potions, and Specials
- Held Cards - Master this unique mechanic (right-click to save cards!)
- Codex System - Find relics and permanent upgrades
- Score System - Learn competitive scoring mechanics
- Strategy Tips - Best practices for survival

### Tutorial Features

Golden Spotlight System:
- Important UI elements highlighted with pulsing golden borders
- Dark overlay creates professional focus mode
- Highlighted elements remain fully interactive (no blocking!)

Interactive Learning:
- Tutorial auto-demonstrates mechanics (e.g., auto-clicks "Draw Room")
- Step-by-step progression with clear visual guides
- Always shows progress and provides skip option

Player-Friendly:
- Skip Anytime with confirmation modal (prevents accidents)
- One-Time Only - Never interrupts experienced players
- Non-Intrusive - Respects player preference

### Keyboard Protection System

During tutorial, keyboard shortcuts are intelligently managed:
- Allowed Keys: Space, Arrow Keys, ESC (for navigation)
- Blocked Keys: D, A, Q, U, S, 1-5 (all game shortcuts)
- Smart ESC: Opens skip confirmation instead of instant close
- Auto-Restore: All shortcuts work normally after tutorial ends

## Getting Started

First-Time Players:
1. Select Easy difficulty (recommended)
2. Tutorial will automatically start
3. Follow the golden spotlights
4. Press Space to advance steps
5. Press ESC to skip (with confirmation)

Experienced Players:
- Tutorial never shows if you've played before
- To replay tutorial: Clear browser localStorage
- Console command: localStorage.removeItem('dungeon_scoundrel_tutorial_completed')
